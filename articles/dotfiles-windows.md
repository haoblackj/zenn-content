# 概要
- Windows をワンライナーで設定する
- ソフトウェアのインストールと設定ファイルの配置を自動化する
- できるだけ PowerShell で実行する

# 環境
- Windows 11
  - Home でも Pro でも動くはず、ARM は未確認
- PowerShell 5.1
  - Windows11 には最初から入っている
- 管理者ユーザー
  - 一部のソフトウェアのインストールに管理者権限が必要
